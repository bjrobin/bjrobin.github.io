字符串切片

除了索引，字符串还支持 切片。索引可以得到单个字符，而 切片 可以获取子字符串:

切片的索引有默认值；省略开始索引时默认为0，省略结束索引时默认为到字符串的结束:

注意切片的开始总是被包括在结果中，而结束不被包括。这使得 s[:i] + s[i:] 总是等于 s
