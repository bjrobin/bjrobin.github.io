注释

Python中的注释以井号 # 开头，并且一直延伸到该文本行结束为止。
注释可以出现在一行的开头或者是空白和代码的后边，但是不能出现在字符串中间。字符串中的井号就是井号。
因为注释是用来阐明代码的，不会被 Python 解释执行。

表达式，运算符+、-、*、/

简单的说，表达式是由运算符和运算对象组成的，括号 (()) 用来分组。

整数类型，浮点数类型

除了 int 和 float，Python也支持其他类型的数字，不用着急，后面会讲到。
